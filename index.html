    Seputar Techno dan Pendaftaran

    TECHNO 2022 merupakan acara kebersamaan dalam rangka menyambut dan melantik mahasiswa baru School of Computer Science (SOCS) BINUS UNIVERSITY 2026. Karena itu, acara ini hanya diselenggarakan dan dapat diikut sekali oleh setiap SOCS BINUSIAN 2026. Acara TECHNO juga dikemas dengan beberapa penampilan, games, bahkan materi-materi pembelajaran melalui PBP dan HIMTI KIT.

    Secara singkat, TIDAK. Anda dapat langsung mendaftar secara gratis. Acara-acara TECHNO sebelumnya memang dikenakan harga tiket masuk (sebesar Rp 150.000) untuk menunjang biaya transportasi dan catering makanan. Sedangkan, TECHNO tahun ini diadakan secara online, sehingga kedua biaya tersebut dapat dipangkas secara penuh.

    Tidak, acara ini juga terbuka bagi mahasiswa dalam setiap jurusan yang berada di dalam naungan fakultas School of Computer Science, yakni:
    1. Computer Science (Regular)
    2. Computer Science - Global Class
    3. Game Application and Technology (GAT)
    4. Mobile Application and Technology (MAT)
    5. Computer Science - Applied Statistics (TI + Stat)
    6. Computer Science - Applied Mathematics (TI + Mat)
    7. Cyber Security
    8. PPTI BCA
    9. Master Track Computer Science
    10. Computer Science - Software Engineering
    11. Data Science

    Sesi Pengenalan Bahasa Pemrograman (PBP)

    Jika Anda merupakan seorang pemula, C dan Python merupakan salah satu bahasa pemrograman yang dapat dipakai untuk memahami prinsip algoritma secara mudah. Namun, jika Anda sudah berpengalaman dalam bahasa pemrograman lainnya, perlu diketahui bahwa bahasa C akan dipakai dalam awal perkuliahan melalui mata kuliah COMP6047 - Algorithm and Programming dan COMP6048 - Data Structures. Begitu pula dengan bahasa Python yang digunakan dalam perkuliahan di BINUS @SENAYAN.

    Sebuah IDE, seperti Dev-C++, merupakan sebuah program editor yang bertujuan untuk mengatur dan mempermudah proses pemgembangan dan pengujian program melalui bahasa pemrograman tertentu. Jika dibandingkan dengan aplikasi seperti Notepad, Dev-C++ dan berbagai IDE lainnya dapat mendeteksi berbagai jenis error seperti syntax error yang dapat mengakibatkan masalah saat program tersebut di-compile.

    Meskipun Anda masih dapat membuat program C menggunakan IDE dan compiler lainnya (seperti Visual Studio, Geany, dan Clang), perlu diketahui bahwa IDE Dev-C++ dan compiler TDM-GCC (terpasang secara default di dalam Dev-C++) akan diwajibkan dalam sesi dan ujian praktikum, sebagaimana ditetapkan oleh Software Laboratory Center (SLC) BINUS UNIVERSITY.

    HIMTI KIT

    HIMTI KIT merupakan sebuah starter kit yang terdiri atas materi perkuliahan dari Semester 1-4 serta software yang akan digunakan selama perkuliahan. HIMTI KIT dapat digunakan sebagai acuan tentang materi-materi yang akan dipelajari nantinya.

    Untuk mendapatkan akses HIMTI KIT, Anda harus dinyatakan hadir dalam acara Main Event TECHNO 2022. Periode absensi akan diumumkan secara langsung oleh panitia saat Main Event berlangsung.
